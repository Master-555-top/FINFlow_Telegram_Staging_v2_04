# FinFlow v3.0 — Foundation v1.1 Latest

Это актуальный пакет для первого запуска нового Mini App.

## Сначала сделать только это

```bash
npm install
npm run dev
```

Открыть:

```bash
http://localhost:3000
```

## Проверить

- нет Hydration failed;
- показывается живое время;
- время суток считается на клиенте;
- работает Dev / Error Log;
- интерфейс открывается нормально.

## Потом

После успешного запуска идём в Step 2 — Daily Money Planner:

- сейчас денег;
- сколько нужно грязными сегодня;
- бензин;
- Drivee 13%;
- рабочий фонд;
- фонд встреч;
- банк;
- машина;
- подушка;
- ходовая;
- мини-цели;
- реалистичность дня.
