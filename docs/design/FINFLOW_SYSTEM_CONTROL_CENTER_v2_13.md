# FINFlow System Control Center v2.13

## Design direction
System is now a compact control center. It should feel like an app settings/control screen, not a technical document.

## Rules
- First level: only section buttons.
- Section detail: back button, short section name, tiny version badge.
- Inner navigation: one-word labels where possible.
- Long text: expandable details only.
- Mobile: no horizontal scroll; bottom nav must not cover important actions.
- Desktop: hover/glow feedback is preserved.

## Current focus
Keep iterating System until the user confirms it feels clean enough, then apply the same design language to Day/Money/Work/Funds/AI.
