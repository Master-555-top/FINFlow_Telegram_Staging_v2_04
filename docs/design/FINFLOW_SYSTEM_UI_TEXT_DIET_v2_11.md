# FINFlow System UI Text Diet v2.11

## Direction
System must feel like a premium mobile command center, not a documentation page.

## Rules
- Root screen: only category buttons + tiny version.
- Detail screen: back button + title + subsection tabs.
- Paragraphs are hidden unless the user asks for details.
- Long technical labels must become short labels: Telegram, Verify, Deploy, Supabase, Cloud.
- Version format inside UI: `v2.11`, not full module chain.
- Keep one color language: black/deep purple glass with controlled cyan/magenta/violet accents.

## Preserved
- Hover glow/outline behavior on desktop.
- Safe-by-design: no raw initData/hash/tokens.
- Cloud writes off until explicit guarded phase.
