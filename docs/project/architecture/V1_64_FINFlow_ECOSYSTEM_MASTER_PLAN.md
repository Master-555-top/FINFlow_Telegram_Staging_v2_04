# v1.64 — FINFlow Ecosystem Master Plan

## Core idea

FINFlow is not just a Telegram Mini App.

FINFlow is a personal operating ecosystem for:
- money;
- taxi work;
- car survival;
- bank transaction review;
- daily planning;
- goals/funds;
- AI decision support;
- learning/career transition;
- security and project continuity.

## Ecosystem modules

### 1. Day Core

Purpose:
- decide what to do today;
- calculate clean money;
- detect impossible plans;
- plan work/rest/tasks.

### 2. Taxi Work Engine

Purpose:
- orders;
- gross turnover;
- Drivee commission;
- Drivee top-up cashflow;
- active hours;
- shift hours;
- zone/time analytics.

### 3. Car Cost and Maintenance Engine

Purpose:
- fuel from odometer;
- consumption range;
- oil reminders;
- maintenance history;
- repair reserve;
- car risk warnings.

### 4. Bank Review Engine

Purpose:
- import bank candidates;
- prevent false income from transfers;
- manually approve/reject/postpone;
- convert reviewed rows to records.

### 5. Funds and Obligations Engine

Purpose:
- car payment;
- bankruptcy payment;
- repair fund;
- cushion;
- relocation;
- girlfriend/work base/birthday;
- flexible mini-goals.

### 6. AI Assistant

Purpose:
- local deterministic advice;
- later external AI server-side;
- answer what to do next;
- prevent unrealistic plans;
- explain money decisions.

### 7. Personal Calendar/Task Planner

Purpose:
- errands;
- girlfriend meetings;
- car service;
- sleep/rest;
- realistic available work window.

### 8. Learning and Exit-from-Taxi Track

Purpose:
- connect finances with learning;
- IT/QA/web/dev/AI path;
- project-based skill development;
- future income transition.

### 9. Security and Ownership Layer

Purpose:
- private repo;
- no exposed keys;
- no public private_raw_data;
- Supabase RLS;
- backup/export;
- anti-copy/IP protection.

### 10. Project Memory and Protocol System

Purpose:
- continuity across chats;
- exact user messages;
- assistant action tracking;
- active task state;
- interruption recovery;
- anti-regression.

## Ecosystem roadmap

### Near-term private MVP

- Daily fuel budget from odometer.
- Real Supabase profile resolver behind flag.
- Cloud sync for records/funds/obligations.
- Telegram launch on real phone.
- Manual bank review persistence.
- Backup/export.

### Mid-term operating system

- Calendar/task planner.
- Better mobile UI.
- Charts/reports.
- AI daily planner.
- Car maintenance assistant.
- Work efficiency analytics.

### Long-term ecosystem

- Multi-project dashboard.
- Learning/career transition tracker.
- Revenue diversification planner.
- Personal AI partner memory across projects.
- Secure deployment and ownership strategy.
