# PRE-START FULL REVIEW STATUS — FinFlow v3.0 v1.4

Дата: 2026-06-15

## Статус

Перед началом активной разработки зафиксирован обязательный этап: **полная ревизия чата, файлов и исторических данных**.

Важная честная пометка: в рамках этого пакета выполнена системная ревизия требований и источников, но **не завершена машинная построчная нормализация всех исторических доходов/расходов/смен**. Это отдельный этап Import/Normalization, потому что старые файлы содержат тысячи строк, разные форматы, банковскую выписку, Telegram-историю, личные данные и смешанные сообщения.

## Что уже включено в v1.4

- Все глобальные требования проекта перенесены в проектные файлы.
- Старые данные признаны обязательным источником для обучения и статистики.
- Добавлен отдельный план нормализации старых доходов/расходов/смен/фондов.
- Добавлен аудит покрытия требований.
- Добавлено правило: **перед каждым большим шагом сверяться с docs/project/**.
- Добавлено правило: **не начинать импорт как "найти цифры и записать" — только через raw → parse → normalize → review → confirm**.

## Что нельзя считать завершённым

- Полный построчный импорт Telegram-истории.
- Полная категоризация всех расходов из истории.
- Полный импорт банковской выписки.
- Точный расчёт исторических средних по бензину, Drivee, заказам, зонам и рабочим расходам.

## Следствие для разработки

Можно начинать приложение с Foundation и Daily Money Planner, но Import Center должен быть спроектирован так, чтобы позже подтянуть **всю старую историю**, а не потерять её.
