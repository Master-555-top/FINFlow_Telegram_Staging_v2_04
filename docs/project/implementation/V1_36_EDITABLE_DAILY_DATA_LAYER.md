# v1.36 — Editable Daily Data Layer

## User requirement

All data except final analytics must be editable manually, conveniently and simply.

## Rule

Editable:
- current money
- date/time
- gross done
- orders done
- expected gross
- target gross
- target clean
- Drivee rate
- fuel planned
- fuel already paid
- active hours
- full shift hours
- day expenses/tasks
- task title
- task amount
- task priority
- include/exclude task
- delete task

Not manually editable:
- final analytics
- averages
- target hit rate
- best/worst day
- recommendations
- calculated clean shift income
- calculated free after plan

These are derived from editable source data.

## Why

The user must always be able to correct the app when real life differs from the app assumptions.
