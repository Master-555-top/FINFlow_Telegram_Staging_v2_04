# v1.38 — Editable Funds & Obligations UI

## Goal

The user must be able to edit all source data conveniently, except final analytics.

v1.38 adds full current-day editing for:
- obligations
- funds
- target amounts
- current saved/current amount
- priority
- participation in today's allocation
- create/delete records

## Product rule

Editable source data:
- obligation title
- obligation due amount
- obligation saved amount
- obligation due day of month
- obligation priority
- fund title
- fund target amount
- fund current amount
- fund priority
- fund can receive today

Derived data:
- allocation result
- shortage
- mode
- strategy
- final analytics
- recommendations

## Why

FinFlow must never trap the user in AI-made assumptions. The user can correct every base value.
