# SECURITY SCAN v1.26

## Статус

v1.26 не добавляет секреты, токены, реальные ключи Supabase, банковские PDF в UI или публичную часть кода.

## Важное правило

`private_raw_data`, банковские PDF, сырые архивы, `.env`, токены и личные данные нельзя загружать в GitHub, публичное облако, Vercel, Supabase Storage или открытые архивы.

## Supabase-ready schema

SQL-файл содержит только структуру таблиц и RLS-шаблоны. Он не содержит реальных ключей, URL, connection string или пользовательских секретов.

## Риск

Локальное сохранение состояния в браузере содержит только redacted excerpts и review metadata. Сырые выписки и приватные источники не должны попадать в localStorage.
